# 外科醫師為什麼該學寫程式?一個大腸直腸外科醫師的跨領域之路

原文網址:https://blog.shihfenghuang.com/why-surgeons-should-code/
發布日期:2026-02-21

---

我是一個每天開刀的大腸直腸外科醫師,同時也會寫 Python、R、SQL、搞統計分析、做機器學習。

每次跟同事提到這件事,得到的反應通常是:「你哪來的時間?」或是更直接的:「學這個幹嘛?」

今天就來回答這個問題。

#### 一切的起點:SPSS的力有未逮

住院醫師時期,我跟大家一樣,做研究就是打開 Excel,手動整理資料、手動跑統計、手動做圖表。

有一次,我用SPSS跑邏輯迴歸,用盡了心血各種調整變數,怎麼樣模型就是無法收斂。

我把我遇到的瓶頸跟當時的ChatGPT(GPT-4o)討論,結論是我想觀察的outcome事件數太少了,用普通的邏輯迴歸非常容易過擬合(overfitting)或是完美分離(perfect separation),係數永遠會不穩定。

當時的GPT建議我,如果要做下去,就得要用R的Firth或是用Python的penalized regression。

這篇統計我已經花了很多時間,真的不甘心就這麼放棄,於是牙一咬心一橫,我就一步一步問到底要怎麼作。

那天就是我的腳嘗試跨領域的第一天。

一個零基礎的人要搞這些東西真的不容易,我一開始什麼概念都沒有,光是設置環境我就搞了老半天。買了一些書自己一步一步啃,從for loops、while loops開始每天練習,enroll了Coursera和DataCamp的線上課程,每天我就這樣土法煉鋼無頭蒼蠅式的學習。

那時剛升主治醫師沒多久,臨床的工作對我來說還是有一定的壓力。這樣兩頭燒的結果,我的睡眠時間被大幅壓縮,現在回頭想想還真不知道到底怎麼撐的。

但是這種學習的感覺真的很快樂。

對我來說那個 Aha moment 就是,當我用R的tableone套件、當我用Python的隨便一個df.describe(),一秒就跑出了以往我用SPSS要花至少半小時以上的基本敘述統計,我就覺得不行這條路我得學下去,因為真的是太酷了。

學著學著,我發現程式碼能做的事遠比我想的多得多。自動化、而且可重現,我很快就發現了這對臨床醫學研究來說是多大的助力。

#### 程式幫我做了什麼?

老實說,學寫程式不會讓你開刀開得更好——至少不會直接讓你的手更穩。但它會讓你在手術「以外」的事情上快非常多,而且錯更少。

【自動化收 data】只要 IRB 通過、和院內 IT 部門溝通過取得權限,你完全可以使用自己寫的 Python 腳本來擷取臨床資料。幾百個病人的資料其實就是一兩小時的事。

【自動化分析流程】以前要跑一份 Cox regression 要多久?打開 SPSS、設定變數、選模型、跑結果、手工刻到 Word 表格。現在我用 Python 寫好 pipeline,丟進資料,幾秒鐘就能跑完所有分析。我甚至為此做了一個 SaaS 統計分析平台叫 Shadex。

【機器學習預測臨床 outcomes】以往我從來不知道機器學習跟臨床醫學可以在這個地方交織在一起。傳統的邏輯迴歸是 inference,而機器學習常是 prediction 為目的,可以預測 complications、blood loss、length of hospital stay 等各種臨床端點。

【深度學習實現 CT 影像自動分析】直腸癌手術難度跟骨盆結構有很大關係。傳統上我們靠肉眼看 CT、手動量測骨盆徑線,花時間、有誤差。我用深度學習自動辨識 CT,一個病人的分析從 15 分鐘縮短到不到 30 秒。

#### 不是要你變成工程師

學寫程式的目的不是要你變成軟體工程師,而是給自己多一項工具。就像你會用超音波、會看 CT,程式也只是另一個工具。差別在於:會用這個工具的外科醫師,目前還非常少。

強大的 AI 語言模型加速了整個學習曲線。你不需要精通語言,你需要能看懂 AI 寫的 code 在做什麼,把整個分析當成一種 Vibe Coding。

#### Surgical Data Science:這是正在發生的趨勢

Data Science 和外科的交集就叫 Surgical Data Science (SDS)。這不是我發明的詞,是國際上快速成長的領域。EAES 已經成立 SDS 工作小組,Annals of Surgery、British Journal of Surgery 近年都大量刊登相關研究。

如果你是正在尋找研究方向的年輕外科醫師,這可能是你最好的差異化機會 —— 大部分外科醫師不會寫程式,大部分會寫程式的人不懂手術。兩邊都懂一點的人站在非常獨特的位置。

#### 寫在最後

我不是天才,程式底子也不是從小打下的。我是當了主治醫師之後才認真學的。學寫程式沒讓我開刀更快,但讓我的研究快了十倍、讓我能問以前問不了的問題。

如果你看完這篇覺得有一點心動,那就去裝一個 Python 試試看吧。
